# Perfect Guess Game
This is a simple number guessing game made using

In this game, the computer randomly selects a number between 1 and 100, and the player has to guess the correct number. After every guess, the program gives a hint whether the number is higher or lower.

I made this project while practicing Python basics like loops, conditions, user input, and the random module.

## Features
- Random number generation
- Higher/Lower hints
- Counts the number of attempts
- Beginner-friendly project

## Technologies Used
- Python

## How to Run
1. Install Python on your system
2. Download or clone this repository
3. Run the Python file

- Working with loops and conditions
- Taking user input in Python
- Using the random module
- Building a simple console-based game
